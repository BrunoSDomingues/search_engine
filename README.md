## Conceitos

### Conceitos básicos

- Limpeza de dados + normalização;
- Queries booleanas;
- Retrieval booleano;
- Ranking tf-idf;

## Execução

1) Rodar o comando `./config.sh`
2) Rodar o comando `./run.sh "query1 (and/or query2 and/or query3)"`.
